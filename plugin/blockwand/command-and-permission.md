# Command and permission

| Command | Permission | Description |
| :---: | :---: | :---: |
| /blockwand give &lt;PlayerName&gt; | blockwand.give | Give a block wand to a specific player. |
| /blockwand reload | blockwand.reload | Reload config file of plugin. |
