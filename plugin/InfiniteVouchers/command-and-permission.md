# Command and permission


| Command | Permission | Description |
| :---: | :---: | :---: |
| /iv | none | Shows all commands |
| /iv list | infinite.vouchers.list | List all available vouchers. |
| /iv editor | infinite.vouchers.admin | Edit settings ingame with a Gui |
| /iv give <player> <voucher> <amount> | infinite.vouchers.give | Give a voucher to a player. |
| /iv giveall <voucher> <amount> | infinite.vouchers.giveall | Give a voucher to all online players. |
| /iv reload | infinite.vouchers.reload | Reload the plugin (configuration and language). |
